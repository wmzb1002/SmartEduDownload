## **v0.0.2**

**2025.9.26**

此次更新内容：

- 添加对输入URL中headers参数的解析支持

- 使用urllib替代requests库发起HTTP请求

- 实现文件流式下载，每次读取1024字节数据

- 支持从URL中自动提取默认文件名

- 增加异常处理机制，提升程序健壮性

- 添加下载完成后的3秒延时自动退出功能
